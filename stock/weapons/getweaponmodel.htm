GetWeaponModel( <weapon name>, <model variant> )

Module: Weapons

Summary:

Returns the name of the weapon model used for the given weapon.



Example:

sniper_model = GetWeaponModel("mosin_nagant_sniper", 0);



Required Args:

  • 1 : <weapon name> The name of the weapon.


Optional Args

  • 1 : <model variant> Model variant (0-15).
