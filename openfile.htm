OpenFile( <filename>, <mode> )

Module: File

Summary:

Open a file for reading, writing, or appending. If successful returns a file number, otherwise returns -1



Example:

filenum = OpenFile( "info.txt", "read" );



Required Args:

  • 1 : <filename> The name of the file to open.
  • 2 : <mode> The file mode. Valid arguments are 'read', 'write' and 'append'.
