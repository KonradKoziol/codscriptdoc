StartTraverseArrival( <yaw angle> )

Module: AI
SP Only

Summary:

Runs traverse arrival animscript on an actor.



Call this on:

<actor> An actor



Example:

soldier StartTraverseArrival( 45 );



Required Args:

  • 1 : <start pos> the offset to start the arrival animation from
  • 2 : <yaw angles> the arrival angle
