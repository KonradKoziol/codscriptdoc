MoveY( <yvalue>, <time>, <acceleration time>, <deceleration time> )

Module: Motion

Summary:

move this entity to the given world y value



Call this on:

script_model, script_origin or script_brushmodel



Example:

hangardoor MoveY( 320, 10 );



Required Args:

  • 1 : <point> The y value to move the entity to, as a floating point number
  • 2 : <time> The time to move the entity in seconds


Optional Args

  • 1 : <acceleration time> The time spent accelerating in seconds
  • 2 : <deceleration time> The time spent decelerating in seconds
